Here are some important terms Jenkins use.

Job/Project: They all refer to runnable tasks that are controlled / monitored by Jenkins

Build is Result of one run of a Project

a build can have a testReport

consoleText is the log for a build

a Node is a logical computer to run Jobs

a Node can have one or more Executor to allow Jobs executed concurrently
