This is docker image I created for including highcharts in zeppelin

zeppelin does not support Highcharts of the shelf
So I developed a library to add Highcharts support
And I call it zeppelin-highcharts

Inspired by one question on stackoverflow
spark-shell also does not plot function
So I add support for spark-shell as well

And I change name from zeppelin-highcharts to spark-highcharts
Since it can be used either on zeppelin or spark-shell

Note: Highcharts is not free for commerical usage
